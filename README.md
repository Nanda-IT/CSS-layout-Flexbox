# Challenge Pemrograman Web - Profil Biodata

Proyek ini adalah implementasi desain web sederhana menggunakan **HTML5** dan **CSS3**. Proyek ini dibuat sebagai bagian dari latihan pengembangan web untuk mereplikasi tata letak (layout) yang terdiri dari header banner, navigasi, sidebar biodata, dan tabel pengalaman kerja.

## 🚀 Fitur
- **Header Banner**: Menggunakan gambar full-width yang responsif.
- **Navigasi**: Menu navigasi yang bersih di bagian atas.
- **Layout Kolom**: Pembagian konten menggunakan Flexbox (Sidebar untuk Biodata dan Konten Utama untuk Pengalaman Kerja).
- **Desain Tabel**: Menampilkan riwayat pekerjaan dalam format tabel yang rapi.
- **Responsif**: Layout dapat menyesuaikan pada layar perangkat yang lebih kecil.

## 🛠️ Teknologi yang Digunakan
- **HTML5**: Untuk struktur konten semantik.
- **CSS3**: Untuk styling, tata letak Flexbox, dan pewarnaan.
